# Charsi or Treasure?

Diablo II: Resurrected quick item lookup — is that drop worth keeping or selling to Charsi?

## Running

```bash
npm install       # first time only
npm run dev       # dev server with hot reload (recommended)
npm run build     # production build
npm run preview   # preview the production build (requires npm run build first)
```

## What it does

- **Search** across all item types: uniques, magic jewels, rainbow facets, base items, and commodities
- Search matches on item name (highest priority), base item name, and description/note
- Color-coded tiers: Godly/GG, High Demand Mid Value, Stash It, Charsi Food, Magic/Rare, Base Items, Commodities
- Tabs for browsing Magic Items, Jewels & Charms, Base Items, and Commodities with price estimates
